小知识讲解：

  • 以上JS代码中的 tables.rows 的 rows：rows 集合返回表格中所有行（TableRow 对象）的一个数组,即一个 HTMLCollection。其语法：tableObject.rows[]
  •  标签：
    • 所有浏览器都支持
    • 定义与用法： 标签定义 HTML 表格。简单的 HTML 表格由 table 元素以及一个或多个 tr、th 或 td 元素组成。
      tr 元素定义表格行，th 元素定义表头，td 元素定义表格单元。
      更复杂的 HTML 表格也可能包括 caption、col、colgroup、thead、tfoot 以及 tbody 元素。
    • 可选的属性如下表所示：
属性 值 描述
align left / center / right 规定表格相对周围的对齐方式
bgcolor rgb(x,x,x) / #xxxxxx / colorname 规定表格的背景颜色
border px 规定表格边框的宽度
cellpading px / % 规定单元边沿与其内容之间的空白
cellspacing px / % 规定单元格之间的空白
width px / % 规定表格的宽度
rules none / groups / rows / cols / all 规定内侧边框哪个部分是可见的
summary text 规定表格的摘要
frame void / above / below / hsides / lhs / rhs / vsides / box / border 规定外侧边框哪个部分是可见的
  • 标签表格主体（正文）。该标签用于组合 HTML 表格的主体内容。
    • 注释： 内部必须拥有 标签！
    • 、 以及 很少被使用，这是因为糟糕的浏览器支持。tbody 元素应该与 thead 和 tfoot 元素结合起来使用。
      thead 元素用于对 HTML 表格中的表头内容进行分组，而 tfoot 元素用于对 HTML 表格中的表注（页脚）内容进行分组。它们的出现次序是：thead、tfoot、tbody。

JQuery 实现方法:

首先引入JQuery库：<script type="text/javascript" src="script/jquery-3.2.1.js" ></script>

将js代码注释掉（当然删除也可以只要你不怕再写一次····），加入以下JQ代码：

$(function(){
     $('#tab tr:nth-child(odd)').addClass('thisline');
})

JQuery :nth-child() 选择器:

  • :nth-child(n) 选择器选取属于其父元素的不限类型的第 n 个子元素的所有元素。
  • 语法：:nth-child(n|even|odd|formula)
参数 描述
n 要匹配的每个子元素的索引。必须是一个数字，第一个元素的索引号是1
even 选取每个偶数子元素
odd 选取每个奇数子元素
formula 规定哪个子元素需要通过公式（an+b）来选取。实例：p:nth-child(3n+2) 选取每个第三段，从第二个子元素开始。
